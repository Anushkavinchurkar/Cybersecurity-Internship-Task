Task 1 - Vulnerability Assessment and Penetration Testing (VAPT)
Objective
To identify open ports, running services, and operating system details using Nmap scanning techniques.

Tools Used
Kali Linux
Nmap

Commands Used
nmap -sV 127.0.0.1
nmap -O 127.0.0.1

Summary
Performed service version detection and OS detection using Nmap. Analyzed exposed ports and evaluated possible security risks.

Outcome
Identified open ports and gathered system information which may increase attack surface if not secured properly.
